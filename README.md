# The Last Blade 2 - Official Artwork Collection

Artwork from SNK's The Last Blade 2 (1998), sourced from the official Neo Geo
website via the Wayback Machine and Imgur archives.

## Contents

- **Character folders** (20 characters) - Main art + concept sketches (genga)
- `weapons/` - Weapon illustrations
- `backgrounds/` - Stage backgrounds from the original SNK site
- `backgrounds_imgur/` - High-res animated stage backgrounds (800x336)
- `openings/` - Opening animation frames
- `menu/` - Character select screen thumbnails
- `promo/` - Promotional images

## Sources

- [SNK Official Last Blade Page (Wayback)](https://web.archive.org/web/20010426060427/http://www.neogeo.co.jp/gextuka/1coin/special.htm)
- [Last Blade Backgrounds (Imgur)](https://imgur.com/a/last-blade-backgrounds-yiGnS)
